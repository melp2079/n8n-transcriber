# n8n Transcriber for Render

🚀 **Ready-to-deploy n8n with FREE audio transcription**

## What's Included
- ✅ n8n latest version
- ✅ FFmpeg for audio processing
- ✅ Free Whisper transcription (no API keys!)
- ✅ Optimized for Render free tier (512MB RAM)
- ✅ 1GB persistent storage

## One-Click Deploy

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/melp2079/n8n-transcriber)

## Features
- Monitor Google Drive for new .m4a files
- Automatically transcribe using local Whisper
- Save transcripts as Google Docs
- Works with your existing email → Drive workflow

## Memory Settings
- Using `whisper-tiny.en` model (smallest footprint)
- Node.js limited to 384MB RAM
- Best for audio files under 5 minutes
